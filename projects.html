Project UNO
Built a prototype of one-handed video game controller designed for people with limited mobility on one hand. The prototype was intended to be used for PC games. It was built on Arduino and used joystick, gyroscope, and buttons as its components to replicate function of a mouse. Worked on functional testing for each component. Team of 5 project.
  • Arduino
Fabflix
A movie database website. Users can login, browse for movies, search for movies, sort results, add a movie to the shopping cart, and checkout. Admin has separate login that allows them to add actors and movies, and view database metadata. (Originally designed to be deployed on AWS, migrated to Heroku) Team of 2 project.
